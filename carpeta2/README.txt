Esta es la Interfaz del Juego Pokemon Pro

Para poder verla necesitas:
En WINDOWS
Ir a la carpeta dist y dar doble click en el archivo PokemonPro.jar, debes 
contar con Java instalado en tu ordenador

En Linux
Debes ir a esta carpeta desde la terminal, desde la misma ingresar a dist y 
ejecutar el jar con el comando 

java -jar <nombre del jar>

debes tener instalado Java en tu ordenador

Se debe de poner el contenido de carpeta 1 y carpeta 2 en una sola llamada PokemonPro
